# 🎮 Ninja Battle Arena - Godot

![Godot](https://img.shields.io/badge/Godot-4.2-blue)

Game pertarungan ninja 2D dibuat dengan **Godot Engine**!

## 👤 Karakter
- 🟠 Naruto | 🔵 Sasuke | 🩷 Sakura | ⚪ Kakashi

## 🎮 Controls
- WASD: Move
- J: Attack | K/L: Skills | U: Ultimate
- Space: Jump | Shift: Dash

## 🚀 Play Online
[Play on GitHub Pages](https://YADI-ganteng.github.io/NinjaBattleArena)

## 🛠️ Open in Godot
1. Download [Godot Engine](https://godotengine.org)
2. Open project.godot
3. Press F5 to run!
